# NATG TTRPG Publisher

A static-site publisher that turns a sanitized Obsidian vault into a public, player-facing encyclopedia for the NATG Star Trek TTRPG campaign.

**Status:** work in progress (post-M1). See [CLAUDE.md](CLAUDE.md) for the working agreement and [specs/](specs/) for design documents.
