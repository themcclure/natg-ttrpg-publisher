# Persephone System

**Primary star:** M4V — Red Dwarf, Main Sequence (flare-active)

---

## Profile

A quadruple red dwarf system. The gravitational environment of four stars means no stable planetary zone exists close in — the bulk of the system is a thick, dispersed debris cloud in the outer circumbinary stable zone. [[outpost-428]] is embedded in that debris cloud.

From inside the system, the sky is never fully dark. The three companion stars appear as reddish points; the sub-giant resolves as a slightly expanded disc. The system is old, dim, and quietly hostile.

The debris cloud provides natural sensor occlusion — an asset for the outpost and a navigational hazard for newcomers. Starfleet-charted approach lanes exist; vessels without that data must come in slow.

The primary star's flare activity is periodic but not precisely predictable. Significant flare events can disrupt communications and complicate EVA operations.
